# 🎉 抽奖系统 - Lottery System

> 一个功能强大、界面美观的多类型抽奖系统，支持九宫格、转盘、刮刮乐、砸金蛋和年会大屏抽奖等多种模式。

## 📁 项目结构

```
lotteryProject/
├── index.html          # 主页（科技风格）
├── README.md           # 项目说明
├── LICENSE             # MIT License
├── config/             # 配置文件夹
│   ├── index.html      # 配置页面
│   └── lottery_config.json  # 奖品配置文件
├── lottery-grid/       # 九宫格抽奖
│   ├── native/         # 原生JavaScript版本
│   └── vue/            # Vue版本
├── lottery-wheel/      # 转盘抽奖
│   ├── native/
│   └── vue/
├── lottery-scratch/    # 刮刮乐抽奖
│   ├── native/
│   └── vue/
├── lottery-egg/        # 砸金蛋抽奖（带锤子特效）
│   ├── native/
│   └── vue/
└── lottery-screen/     # 年会大屏抽奖（支持一次抽多人）
    ├── native/
    └── vue/
```

## ✨ 功能特性

### 🎯 抽奖类型
- **🎰 九宫格抽奖**：经典的九宫格布局，点击格子抽奖
- **🎡 转盘抽奖**：旋转转盘，指针指向奖品
- **🎟️ 刮刮乐抽奖**：模拟刮刮乐，刮开涂层看结果
- **🥚 砸金蛋抽奖**：带真实锤子特效的砸金蛋体验
- **📺 年会大屏抽奖**：支持一次抽多人，适合大型活动

### 🎨 界面设计
- **🌟 科技风格**：现代科技感的UI设计，渐变背景和动画效果
- **📱 响应式布局**：适配各种屏幕尺寸，从手机到桌面
- **💫 流畅动画**：精美的过渡效果和交互动画
- **🎭 主题配色**：蓝色和青色的科技感渐变配色

### 🚀 核心功能
- **🎁 奖品管理**：通过JSON配置文件管理奖品信息
- **📊 中奖记录**：详细的中奖历史记录
- **🔧 自定义设置**：可配置奖品概率、数量等参数
- **👥 多用户支持**：年会抽奖支持一次抽多人
- **🔨 锤子特效**：砸金蛋抽奖的真实锤子动画

### 🛠️ 技术栈
- **前端**：HTML5, CSS3, JavaScript (ES6+)
- **框架**：Vue.js 2.2.2（通过CDN）
- **样式**：CSS3 Flexbox, Grid, 动画
- **数据**：JSON配置文件
- **响应式**：媒体查询，自适应布局

## 🚀 快速开始

### 1. 本地运行

```bash
# 方法1：使用Python内置服务器
python -m http.server 8000

# 方法2：使用Node.js (如果已安装)
npx serve

# 方法3：使用PHP内置服务器
php -S localhost:8000
```

然后在浏览器中访问：`http://localhost:8000`

### 2. 配置奖品

编辑 `config/lottery_config.json` 文件，自定义奖品信息：

```json
{
  "prizes": [
    {
      "id": 1,
      "title": "一等奖",
      "image": "奖品图片URL",
      "count": 1,
      "probability": 5
    }
    // 更多奖品...
  ],
  "participants": [
    "参与者1",
    "参与者2"
    // 更多参与者...
  ]
}
```

## 🎮 使用方法

1. **选择抽奖类型**：在主页选择你想要的抽奖模式
2. **配置设置**：根据需要调整抽奖参数
3. **开始抽奖**：点击开始按钮，等待结果
4. **查看记录**：右侧面板显示中奖历史
5. **重置抽奖**：随时可以重置抽奖状态
